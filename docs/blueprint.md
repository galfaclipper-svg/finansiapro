# **App Name**: FinansiaPro

## Core Features:

- Smart Transaction Input: A unified interface for entering cash-in and cash-out transactions, intelligently categorizing and preparing data for accounting processes with single entry.
- Automated Financial Reporting Tool: An intelligent tool that automatically generates and updates general journal, general ledger, income statement, balance sheet, and cash flow statements in real-time, performing complex accounting derivations from transaction inputs. It utilizes a comprehensive Chart of Accounts (COA) specifically tailored for online trading businesses (e.g., Shopee, TikTok, Instagram, WhatsApp), including specialized accounts for employee receivables, bank interest expenses, other income, bank interest income, other expenses, and various promotional/endorsement costs, grouped by their respective COA categories.
- Inventory Management: Automatically tracks and updates merchandise inventory stock based on relevant purchase and sales transactions.
- Interactive Financial Dashboard: A modern, attractive, and dynamic dashboard visualizing key financial performance indicators with customizable time range filters for insightful analysis.
- Professional Export & Print: Export all financial reports to XLXS format, including integrated formulas and print-ready, and generate professional UI PDF versions for printing.
- Company Customization & Data Tools: Manage company details like name, address, and logo. Provides utilities for XLXS data import, smart transaction editing/deletion, and a full data reset function.
- Multi-language Support: Allows users to switch the application's interface language between Indonesian (default), English, and Arabic, catering to a diverse user base.
- Cross-Platform Compatibility (Android & iOS): The application will be developed to be fully compatible and deployable on both Android and iOS mobile platforms, ensuring a wide reach and accessibility for users.
- Smart Transaction Scan and Categorization: An AI-powered tool that automatically reads uploaded proof of transfer documents (e.g., images of receipts), identifies transaction types (cash-in/cash-out), extracts relevant details, and intelligently categorizes them into the appropriate comprehensive Chart of Accounts (COA) for streamlined accounting.

## Style Guidelines:

- Primary color: A sophisticated, trustworthy blue (#247BA0), chosen to convey professionalism and clarity, making financial data feel grounded and precise.
- Background color: A very light, crisp blue-grey (#F0F6F8), offering a clean and unobtrusive canvas that highlights the financial data without visual clutter, appropriate for a professional setting.
- Accent color: An insightful indigo (#4B5FC7), used sparingly for highlights, calls-to-action, and interactive elements to provide visual emphasis and a modern edge while remaining analogous and complementary.
- Headline and body font: 'Inter' (sans-serif), selected for its modern, neutral, and highly readable characteristics, ideal for displaying detailed financial figures and reports clearly and consistently.
- Utilize clean, modern line-style icons that align with financial themes (e.g., charts, ledger books, currency symbols) ensuring visual consistency and intuitive navigation throughout the application.
- Implement a structured, spacious, and modular layout with a prominent sidebar for navigation, a main content area for detailed reports and dynamic dashboards, optimized for data readability and user workflow efficiency across various screen sizes, including a flexible, responsive design for mobile (Android/iOS) platforms.
- Incorporate subtle, functional animations for seamless transitions between states, loading data, and hover effects on interactive elements, enhancing the user experience without causing distraction.
